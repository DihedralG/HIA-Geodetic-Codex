💠 The ChiR Image Provenance Protocol (ChiR-IPP)

ChiR-IPP is an open, verifiable standard for tagging AI-generated and human-augmented media with provenance glyphs, creating a permanent metadata layer of accountability and trust.

🔐 Why It Matters

  • 🔎 Traceability: Glyphs allow visual content to be mapped to verified metadata across time.
  • 🌐 Trust at Scale: Builds a common foundation for science, art, AI development, and Indigenous sovereignty.
  • ⚖️ Ethics: Establishes user-informed consent and meaningful participation across systems.

📐 Glyph Structure

Each glyph is a visual signature tied to:

  • Object type (e.g., photo, quote, paper)
  • Timestamp & location
  • Source intention & access tier
  • Corner Glyph ID and analysis prompt
