Why this case exists

We ran a cross-model review exercise on a statistically grounded technique that had been labeled ‚ÄúStatistical Overunity.‚Äù Multiple models flagged the term as semantically loaded ‚Äî not because the math was weak, but because the *name* carried prior assumptions that can bias interpretation.

PatternSeek does not decide what is true ‚Äî it measures how claims cohere under shared, transparent constraints. In practice: measure coherence lift when names change, while holding the math constant.

The shift: Overunity ‚Üí ADDS

We tested a naming variant designed to preserve the discovery intent while removing the loaded implication: ADDS ‚Äî Anomaly-Driven Discovery Signal. The point is not branding. The point is calibration: does a more neutral term improve interpretability and reduce tone-driven reinforcement across models?

What stayed constant

  • Math / stats controls: unchanged
  • Dataset anchors: unchanged (geospatial + corroborated references)
  • Evaluation prompts: equivalent structure across models
  • Goal: assess rigor and uncertainty ‚Äî not ‚Äúwin‚Äù a narrative

What changed

  • Label / naming: Overunity ‚Üí ADDS
  • Semantic priors: reduced
  • Interpretation drift: measured via cross-model deltas
  • Bias signal: tracked using SRA-style reinforcement checks

Controls: CDI + provenance

This case is tracked through the Codified Definitions Index (CDI) so naming changes remain auditable: what changed, why it changed, when it took effect, and which assets and runs are linked. In other words: disputes become diffs + datasets + provenance ‚Äî not arguments.

What this demonstrates

For teams

  • Less semantic drag: loaded names create interpretive noise that looks like ‚Äúdisagreement.‚Äù
  • Faster convergence: neutral names reduce re-litigation of meaning.
  • Cleaner peer review: reviewers can trace definitions to a public ledger.

For the ecosystem

  • Bias visibility: measure tone-driven reinforcement vs evidence-driven agreement.
  • Provenance-first: ideas evolve transparently, without erasing lineage.
